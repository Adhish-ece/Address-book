# Address-Book

A simple C-based address book application for storing, retrieving, and managing contact information in a console-based environment.

## Features
- Add new contacts
- Edit existing contacts
- Delete contacts
- Search contacts
- List all contacts
- File-based data storage for persistence

## Installation & Setup

### 1. Clone the repository
```bash
git clone https://github.com/Adhish-ece/Address-Book.git
cd Address-Book
```
### 2. Compile the program
```
gcc -o addressbook main.c contact.c file.c
```
### 3. Run the program
```
./addressbook
```

## Usage
After launching the program, you will see a menu like this:

1. Add Contact  
2. Edit Contact  
3. Delete Contact  
4. Search Contact  
5. List All Contacts  
6. Exit  

Choose any option and follow the on-screen instruction.
